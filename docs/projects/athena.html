Athena

Learning and assessment platform: mock exams, progress tracking, learning analytics, and AI-assisted study support.

Overview

Athena is a learning and assessment platform focused on retention and measurable progress. It combines adaptive practice, analytics, and targeted feedback to help learners improve efficiently.

Approach

Athena explores: how can memory and feedback loops improve retention and make learning measurable?

  • Adaptive mock exams tuned to learner progress.
  • Tracking and analytics that make retention visible over time.
  • AI-assisted hints and explanations linked to a knowledge base.

Architecture highlights

Designed to separate content, assessment logic, and analytics. Integrates with memory stores to tailor practice based on past performance.
